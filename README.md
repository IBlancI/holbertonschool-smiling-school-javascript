# Smiling School - Holberton School Project

Welcome to the **Smiling School** project! This repository contains the source code for a responsive, dynamic school website built from scratch using **Bootstrap 4**, **XML/HTML5**, **CSS3**, and **jQuery**.

## Project Overview

The goal of this project is to implement a pixel-perfect, responsive web application that works seamlessly across Desktop, Tablet, and Mobile devices.

### Key Features
*   **Responsive Design**: Fully adaptable layout for all screen sizes.
*   **Bootstrap 4 Integration**: profound use of the Bootstrap grid system and components.
*   **Custom Styling**: Tailored CSS to match the exact design specifications.
*   **Dynamic Components**: Interactive elements powered by jQuery (Carousels, Dropdowns, etc.).
*   **Accessibility**: Semantic HTML markup.

##  Repository Structure

The project is divided into several HTML pages and asset folders:

*   **`0-homepage.html`**: The main landing page.
*   **`0-pricing.html`**: The pricing and subscription page.
*   **`0-courses.html`**: The course listing and search page.
*   **`fonts/`**: Custom font files.
*   **`images/`**: All static assets/images used in the project.
